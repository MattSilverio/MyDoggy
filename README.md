# MyDoggy

## Como visualizar o projeto?
Ao clonar o repositório, vá ao terminal de sua máquina e digite 'npm install' ou 'yarn install' caso tenha o yarn instalado

## Como funciona o projeto?
O projeto consome uma API com um repositório de fotos de cachorros, a finalidade dele é de ser uma rede social para postar fotos de seus bichinhos. Você pode criar o seu cadastro, criar posts e comentar nas fotos

## Tecnologias Usadas
React , Javascript , CSS
